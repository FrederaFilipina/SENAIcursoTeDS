10. Jogo do Par ou Ímpar

Descrição: O usuário joga contra o computador para ver quem vence no jogo de par ou ímpar.

Como Funciona:
Entradas: O usuário escolhe “Par” ou “Ímpar”. Depois, o usuário digita um número de 0 a 10.

Processamento: O computador gera aleatoriamente um número de 0 a 10.
O sistema soma os dois números.

Saída: Se a soma for par e o usuário escolheu “Par”, ele ganha.
Se a soma for ímpar e o usuário escolheu “Ímpar”, ele também ganha
